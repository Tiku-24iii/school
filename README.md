# 🎓 Exam Management System

A simple web application to manage student exams, results, and attendance.

## 🚀 Features
- Add exams
- Enter marks
- View results
- Attendance system

## 🛠️ Tech Stack
- HTML
- Tailwind CSS
- JavaScript
- Node.js
- MongoDB

## ▶️ How to Run
1. Start backend (server folder)
2. Open client/index.html
